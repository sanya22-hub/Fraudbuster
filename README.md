# FraudBusters 
**Credit Card Fraud Detection **

Goal:To explore, model, and evaluate fraud detection on highly imbalanced transactions.

---

## Highlights
- **EDA**: tidy summaries, outliers (Z-score/IQR), visual trends (uni/bi/multivariate).
- **Prep**: missing-value **imputation**, **StandardScaler**, sampling strategies.
- **Imbalance**: **SMOTE** (train-only) + PR-AUC/Recall-first evaluation.
- **Models**: Logistic Regression, **SVM**, Decision Tree, **Random Forest**, **XGBoost** (+ tuning).
- **Reports**: Confusion matrix, ROC & PR curves, feature importance.
